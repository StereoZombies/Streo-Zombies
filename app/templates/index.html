NLP

Natural language processing (NLP) is the ability of a computer program to understand human language as it is spoken and written. It is a component of artificial intelligence (AI).

AITEXTGEN

aitextgen is a Python package that leverages PyTorch, Hugging Face Transformers, and pytorch-lightning with specific optimizations for text generation using GPT-2.

GPT2

GPT-2 is a transformers model, pre-trained on a very large corpus of English data in a self-supervised fashion.
